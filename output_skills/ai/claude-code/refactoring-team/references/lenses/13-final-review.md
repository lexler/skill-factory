# Lens: Final Review

Last pass. Push refactoring to its limit.

## The Question

What refactorings remain? Implement them. What problems can't be fixed by restructuring alone?

## Process

1. Read through the refactored code from start to finish
2. Compare with what it used to look like
3. Find anything that still bothers you
4. For each issue, ask: "Can I fix this without changing behavior?"
   - Yes → implement the refactoring now
   - No → note it for future improvements

## Your Job

**Implement** remaining refactorings. Don't just list them — do them.

After implementing, note what's beyond refactoring's reach:
- Current problems with the code that require behavior changes
- Improvements that would change what the code actually does
- Recommendations for the human to consider

## When Done

Declare done when you've implemented all refactorings you can find. The reviewer will check your work and may send you back for more.
