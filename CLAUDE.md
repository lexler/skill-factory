# Skill Factory

This repository is a skill creation system for Claude Code.

## Documentation

The `docs/` folder contains all the knowledge needed to create Claude Code skills:
- `docs/knowledge/anthropic-skill-docs/` - Official Anthropic documentation on creating skills
- `docs/project.md` - Information about this skill-factory project
- `docs/map.md` - Repository structure overview

When creating new skills, refer to the documentation in `docs/` for best practices, patterns, and guidelines.

## Output

Created skills are stored in `output_skills/`, organized by skill name.
