# Anthropic Skill Documentation

Official documentation for creating Claude Code skills.

## Documents

**overview.md**
High-level introduction to agent skills. What they are, why they exist, and core concepts.

**skills.md**
Detailed guide on implementing skills in Claude Code. Syntax, structure, and usage patterns.

**best-practices.md**
Proven patterns and common pitfalls. Guidelines for writing effective, maintainable skills.

**quickstart.md**
Quick start guide to get your first skill running quickly.

## Updating

Run `uv run update_docs.py` from this directory to fetch the latest versions from `sources.txt`.
