# Anthropic Skill Documentation

Official documentation for creating Claude Code skills.

```
anthropic-skill-docs/
├── map.md                # This file
├── sources.txt           # URLs to fetch from
├── overview.md           # High-level intro to agent skills
├── skills.md             # Detailed implementation guide
├── best-practices.md     # Patterns and pitfalls
└── quickstart.md         # Quick start guide
```

## Updating

Run `uv run scripts/fetch_anthropic_skill_docs.py` from repository root to fetch latest versions.
